API

hlathena can be used as a python package. Import hlathena by:

import hlathena as ha

Plotting Tools

hlathena.plotting.plot_length(pep_df[, ...])

Plot the distribution of peptide lengths.

hlathena.plotting.plot_logo(pep_df[, ...])

Plot the sequence logo for a given allele and peptide length.

hlathena.plotting.plot_umap(umap_embedding_df)

Plot the UMAP for a given UMAP embedding dataframe.

hlathena.plotting.plot_clustered_umap(...[, ...])

Plot the clustered UMAP for a given UMAP embedding dataframe.

Peptide Projection Tools

hlathena.peptide_projection.PCA_encode(...)

Encodes peptides and performs PCA.

hlathena.peptide_projection.get_umap_embedding(...)

Create UMAP embedding dataframe for peptides.

hlathena.peptide_projection.get_peptide_clustering(...)

Cluster peptides using DBSCAN algorithm on the UMAP embedding.

Annotation Tools

hlathena.annotate.list_tcga_expression_references()

List available TCGA expression reference datasets

hlathena.annotate.get_reference_gene_ids(pep_df)

Method to annotate peptides with gene IDs.

hlathena.annotate.add_tcga_expression(...[, ...])

Add column with TCGA reference expression data.

Prediction Tools

hlathena.predict.predict(model_path, peptides)

Predict the scores of the given peptides using a saved PyTorch model.

Amino Acid Featurization Tools

hlathena.amino_acid_feature_map.AminoAcidFeatureMap([...])

Creates an amino acid feature mapping from a list of amino acid property tables.
