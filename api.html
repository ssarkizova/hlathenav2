API

hlathena can be used as a python package. Import hlathena by:

import hlathena as hat

Analysis Tools

hlathena.plotting.plot_length(pep_df[, ...])

Plot the distribution of peptide lengths.

hlathena.plotting.plot_logo(pep_df[, ...])

Plot the sequence logo for a given allele and peptide length.

hlathena.plotting.plot_umap(umap_embedding_df)

Plot the UMAP for a given UMAP embedding dataframe.

hlathena.plotting.plot_clustered_umap(...[, ...])

Plot the clustered UMAP for a given UMAP embedding dataframe.
