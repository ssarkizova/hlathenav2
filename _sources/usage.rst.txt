Usage
=====

Getting started with HLAthena
------------

To use HLAthena, clone it from the GitHub repository and install with pip:

.. code-block:: console

   (.venv) $ git clone https://github.com/ssarkizova/hlathenav2.git
   (.venv) $ cd hlathenav2/
   (.venv) $ pip install .


Example
------------

For an example workflow, please visit the `tutorial notebook <https://github.com/ssarkizova/hlathenav2/blob/main/notebooks/tutorial.ipynb>`_.


.. note::
