Usage
=====

Getting started with HLAthena
------------

To use HLAthena, clone it from the GitHub repository and install with pip:

.. code-block:: console

   (.venv) $ git clone https://github.com/ssarkizova/hlathenav2.git
   (.venv) $ cd hlathenav2/
   (.venv) $ pip install .

.. note::

   This project is under active development but will soon be uploaded to PyPi and will be pip-installable directly from the command line.
