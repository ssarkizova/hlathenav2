hlathena.plotting.plot_umap

hlathena.plotting.plot_umap(umap_embedding_df: DataFrame, clustered: bool = False, label_col: str | None = None, title: str | None = None, save_path: str | None = None, countplot_log_scale: bool = False) → None

Plot the UMAP for a given UMAP embedding dataframe.

Parameters:

  • umap_embedding_df – A dataframe with the UMAP features. Columns ‘seq’ (peptide sequences),’umap_1’, and ‘umap_2’ are expected.

  • clustered – Indicates whether UMAP embedding has a ‘cluster’ column.

  • label_col – The name of a column denoting the peptide category for comparison. Default value is None, in which case no category comparison will be performed.

  • title – Optional plot title, defaults to None.

  • save_path – Path where that figure will be saved to. Default is None.
